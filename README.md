ember-component-info
==============================================================================

This is proof of concept, how to generate documentation for component .js/.hbs files.
http://localhost:4200/component-info

Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above
